# Task 15 - End-to-End Machine Learning Pipeline

## Objective
Build a complete ML pipeline using preprocessing and model training in a single workflow.

## Dataset
Breast Cancer Dataset (sklearn)

## Steps Performed
- Loaded dataset
- Identified numerical features
- Applied scaling using ColumnTransformer
- Built Pipeline with preprocessing + Logistic Regression
- Trained model
- Evaluated using Accuracy, Precision, Recall, F1-score
- Saved trained pipeline as .pkl file

## Evaluation Metrics

Accuracy: 97%
Precision: 98%
Recall: 97%
