# Task 7: Titanic Survival Prediction using Logistic Regression

## Objective
To predict passenger survival using Logistic Regression.

## Dataset
Seaborn Titanic Dataset

## Steps Performed
- Data cleaning and preprocessing
- Handling missing values
- One-hot encoding
- Feature scaling
- Train-test split
- Logistic Regression training
- Model evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
- ROC Curve and AUC Score

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
